#Buscador de Endereços

## Objetivos
Desenvolver uma aplicação que permita ao usuário buscar endereços. Ele pode fazer isso digitando somente o CEP, 
ou digitando informações como sigla do estado, cidade e bairro. Também é gerado um histórico de buscas


## Fluxo
Basicamente a aplicação pergunta se o cliente deseja realizar a busca por CEP ou por endereço. 
Depois pede o CEP ou os componentes do endereço
Pra finalizar(ou não), pergunta se quer fazer outra busca ou sair
